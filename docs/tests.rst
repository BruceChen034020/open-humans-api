Development & Testing
*********************

``open-humans-api`` is under active development and we have implemented a couple
of tests to check that changes don't break anything.

If you want to contribute
`you can do so on GitHub <https://github.com/openhumans/open-humans-api>`_ and
`find us on Slack <http://slackin.openhumans.org>`_.

Installing ``ohapi`` for development
------------------------------------

To install a development version of this package run the following steps on your
shell:

.. code-block:: Shell

  git clone git@github.com:OpenHumans/open-humans-api.git
  cd open-humans-api
  pip install -e.
